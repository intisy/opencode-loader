# OpenCode Launcher - Specifications & Test Requirements

## Goal
Centralized launcher, project manager, and marketplace for OpenCode.

## Requirements
- [ ] **API Exposure**: Must provide an API for other plugins to consume and integrate with.
- [ ] **Project Picker**: Fully functional, robust project picker and launcher.
- [ ] **Marketplace**: Dedicated 'MCP' and 'Plugins' sections, each featuring a marketplace UI.
- [ ] **CLI Replacement**: Allows the 'oc' prefix command to act as a complete, drop-in replacement for the standard 'opencode' command.
